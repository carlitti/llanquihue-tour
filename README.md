# LlanquihueTourApp

## Descripción
Aplicación en Java que permite leer datos desde un archivo de texto (.txt), crear objetos y almacenarlos en un ArrayList. Luego se muestran y filtran los tours según el precio.

## Estructura del proyecto
- model: clase Tour
- data: clase GestorDatos (lectura del archivo)
- ui: clase Main
- resources: archivo tours.txt

## Ejecución
1. Abrir el proyecto en IntelliJ IDEA
2. Ejecutar la clase Main
3. Ver resultados en la consola
